# EQ2 Harvest Tracker

Track your harvesting adventures.

![Screenshot](https://imgur.com/42BBfo0.png)

### Requirements

- Node
- Yarn

### How to run

Install dependencies:

```sh
yarn
```

Run the program and point to your log file:

```sh
yarn run start C:\EQ2\logs\server\eq2log_MyCharacterName.txt
```
